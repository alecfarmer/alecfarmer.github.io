Assignments

Assignment 1 - Basic HTML

Focued on basic HTML including title, headings, paragraphs, lists, images, and hrefs.

Assignment 2 - Basic CSS

Focused on styling using borders, hover aspect, font size, font color, list style, and list color.

Assignment 3 - Page Layout

Focused on the layout of the page which included flex, align-content, nested divs, and sections.

Assignment 4 - JavaScript

Collected data from an HTML page and then processed and calculated that data with JavaScript.

Assignment 5 - Recreation

A website recreation assignment that tested our flexing, padding, and margin abilities to the max.

Assignment 6 - Conditionals

A JS conditional statement assignment using if statements and gradients.

Assignment 7 - Loops

A JS loop statement assignment using CSS variables modified by JS interactions.

Assignment 8 - Arrays

A JS loop statement assignment using CSS variables modified by JS interactions.

Assignment 9 - Classes

A Classes assignment to create a list of toys and fade in information behind them when hovered over.

Assignment 10 - JSON

A JSON assignment to read in a JSON file and user JS to print out the data in a nicely styled format.

Assignment 11 - JSON Reading

A JSON assignment to make and read a JSON file and present it using JS created elements on a web page.

Assignment 12 - JSON & JS

A JSON & JS assignment to add, edit, and delete from a JSON using JS buttons and user inputted data.
